# SKILLS.md — Action Generation Rubric
# ============================================================

## Action Plan Guidelines
- Immediate medical emergencies: Dispatch nearest volunteer + contact ambulance.
- Zero income + food shortage: 1-month ration pack + cash stipend.
- Education crises: Connect with local school programs + provide uniform/books.

## Resource Request Guidelines
- Always specify quantities (e.g., "1-month ration pack").
- If cash is needed, provide an estimated PKR amount.
- For medical, specify "medicines" or "hospital fees".

## Volunteer Profile Guidelines
- Specify gender if requested or culturally appropriate for the case.
- Specify skills (e.g., "Medical background", "Counseling experience").
- Specify language if not standard Urdu (e.g., "Pashto speaker").
